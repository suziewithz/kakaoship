#kakaoship

Installation
--------------
You will need to install the following

  - python 2.7.9
  
Getting Started
--------------

  - Run analyzer.py
```
[kakaoship] $ python analyzer.py [log_path]
ex) python analyzer.py kakaotalk.txt
```
